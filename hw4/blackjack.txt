This is a blackjack program. J is 11, Q is 12, K is 13, and you can choose between 1 and 11 for A. The user gets random cards, and user can choose whether to get more cards or not. If the sum of all the cards exceeds 21, user looses. If user decideds not to draw any more cards. On dealers turn, dealer also gets random cards. If dealer gets less number than the user or exceeds 21, the dealer loses.
